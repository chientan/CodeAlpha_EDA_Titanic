# Titanic Exploratory Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis on the Titanic dataset to
understand patterns and factors affecting passenger survival.

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Key Analysis Performed
- Data structure exploration
- Missing value detection and handling
- Gender vs Survival analysis
- Statistical validation using groupby
- Data visualization using charts

## Key Insights
- Female passengers had a higher survival rate than males
- Passenger class influenced survival chances
- Missing values were handled appropriately

## Conclusion
The analysis highlights important factors affecting survival on the Titanic and
demonstrates a complete EDA workflow.
